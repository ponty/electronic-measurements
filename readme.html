About¶

Electronic measurements using Arduino, some additional hardware, USB port, Python libraries.

The logic is programmed in Python, which is good for experimenting.

Links:
  • home: https://github.com/ponty/electronic-measurements
  • documentation: http://ponty.github.com/electronic-measurements

Install¶

  • build additional hardware for the project
  • install softusbduino (Python library, firmware in Arduino, V-USB hardware)
  • install pandas
